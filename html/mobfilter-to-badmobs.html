Mobfilter to BadMobs Converter

Convert Fabric Mod Configs to Forge

This tool helps Minecraft modders migrate from the Fabric mod ecosystem to Forge by converting Mobfilter YAML configurations to BadMobs TOML format.

The converter specifically focuses on mob spawn restrictions, creating a BadMobs configuration that disables normal spawning and mob spawners for any entity that was blocked in your original Mobfilter setup.

Conversion Summary

Your BadMobs TOML file has been generated with the following settings for each disabled mob:

  • allowNormalSpawning = false - Prevents natural spawning in the world
  • allowSpawners = false - Prevents spawning from mob spawner blocks
  • allowSpawnEggs = true - Still allows using spawn eggs (for creative purposes)
  • allowConversions = false - Prevents mob conversions (like villagers to zombie villagers)

You may want to edit these settings manually if you need different behavior for specific mobs.
